What’s the scenario?

A show-and-tell evening or an evening when a presenter has come and handed over a set of images on a memory stick. Not in a Powerpoint or similar file.

Step 1: Download Osprey Slideshow

This is the link to download the zip file containing the files; alternatively you could run it direct from the web at xxxxx. But often in Church halls there’s not a lot of internet access so migtht as well download it.

Step 2: Unzip the file

Windows/Linux etc now all can unzip these files – right click in the Windows explorer and extract the files.

Step 3: Double click on index.html

… to load into your browser. This is either Chrome, Firefox or….

Step 4: Click Add files

Click on the Add files button… all the files are loaded in.

Step 5: Click Start Slideshow

Click start slide show (useful to hit F11 to go full screen first)

Arrow keys, page up page down, or any typical remote pointer to advance. Escape stops the show.
